# cloud wallet

## Quick Start

To run this app, clone the repository and install dependencies:

```bash
 npm install
```

export ENV
```
export MONGO_URL=""
export GOOGLE_CLIENT_ID=
export GOOGLE_CLIENT_SECRET=
export FRONTEND_URL=
export JWT_SECRET=
export SOCKET_URL=
export PORT=8081
export CHAIN_PREFIX=
FB_CLIENT_ID=
FB_CLIENT_SECRET=
```

Start the server.

```bash
npm start
````

Navigate to [`http://localhost:8081`](http://localhost:8081)
