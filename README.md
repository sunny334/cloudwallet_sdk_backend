# cloud wallet

## Quick Start

To run this app, clone the repository and install dependencies:

```bash
 npm install
```

export ENV
```
MONGO_URL=""
GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=
FRONTEND_URL=
JWT_SECRET=
SOCKET_URL=
PORT=8081
CHAIN_PREFIX=
FB_CLIENT_ID=
FB_CLIENT_SECRET=
```

Start the server.

```bash
npm start
````

Navigate to [`http://localhost:8081`](http://localhost:8081)
